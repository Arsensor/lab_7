Реализация программы, для вывода информации о продуктах.

Необходимо перейти на сайт Dymmy JSON в раздел DOCS, справа разделы, нам нужен Products. Запрашиваем список продуктов и вывводим весь список. API – стандартизированный протокол, для взаимодействия между программами (В контексте интернета). End point – точка назначения. Необходимо вывести картинку, название, цену, категорию.
